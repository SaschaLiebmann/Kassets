Kassets Command Samples.

In this sample, the usage of Commannd is demonstrated.

CommandBasics.scene

1. The Button in this scene had referenece to PrintLogCommand.asset (A ScriptableObject instance of PrintLogCommand).
2. Execute is fired from the Button's OnClick event. This can be done directly on Hierarchy without creating any script.
3. A Debug Log will be printed on the Console. Text to log can be customized from PrintLogCommand.asset through the Inspector window.
