Kassets Collection Samples.

In this sample, the usage of Collection is demonstrated.

CollectionBasics.scene

1. There are 3 buttons on the Scene:
  a. "Add Grid" to add collection element.
  b. "Remove Grid" to remove collection element.
  c. "Add Value to Random Grid" to add value to random collection element.
2. This sample demonstrate subscription to Add, Remove, and Value change on each component within int collection.
